* It's a plantform game 一个平台跳跃类游戏. * A/D to Move left/right ,W to Jump 玩家通过A/D按键控制主角左右移动, W键控制跳跃. * 让主角借助一个个反方向旋转的钟表(去抓钟表的分针) 努力的向上跳跃. * 主角落在某个钟表上之后 钟表开始消失, 玩家要在消失前努力的跳跃到另外的钟表上. * 游戏中有背景音乐. 每当到达音乐的节拍点时(玩家所在的钟表会随着节拍闪动), 玩家跳跃的能力会有所提升. * 游戏除了努力的在尽可能短的时间攀登到顶端这个游戏目的之外, 还有一个更有挑战,也更有内涵的任务: 去收集隐藏在钟表里的"记忆的私语". * 本游戏基于HTML5技术, 请使用 chrome 或 safari 浏览器的最新版. 在线体验地址 : http://tryhtml5.sinaapp.com/wom/play.html
